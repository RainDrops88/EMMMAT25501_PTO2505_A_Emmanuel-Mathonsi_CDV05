## Objectives

- Create a clean and professional design.
- Ensure easy navigation and accessibility.
- Use responsive design to make your site look good on all devices.
- Include at least 2 projects.
- Provide a brief description, technologies used, and a link to the project or its repository.

---

## Technical Requirements

- Use HTML, CSS, Javascript and React to build your site.
-  Create components for Header, Home, About, Projects and Footer.
- List all ptojects on a separate data.js file.
- Create function to render the projects.
- Attach your pdf resune.

## Project Setup Instructions

  1. Clone the project folder.
  2. Open Commad Terminal and type:
        - git clone, followed by the project name.
        - cd, .....project name.
        - code . 

  3. Install dependencies using:

    ```bash
    npm install
    ```

  4. Run the development server with:

    ```bash
    npm run dev
    ```

  5. Open http://localhost:5173 in your browser to view the app.
